# TEKSPICE
NanoTekSpice est un simulateur de circuits logiques numériques écrit en C++. Il permet de créer, simuler et tester des circuits à partir de fichiers de configuration .nts, en prenant en charge des composants simples (portes AND, OR, XOR, NOT), des composants avancés (compteurs, bascules, registres) et des entrées/sorties spéciales (input/output).
